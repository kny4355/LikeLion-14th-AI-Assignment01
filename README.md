# LikeLion-14th-Assignment-Template
🦁 SKHU 멋쟁이사자처럼 14기 과제 PR 템플릿 레포지토리입니다.
1. 선택한 서비스
넷플릭스(Netflix)의 개인화 콘텐츠 추천 시스템
2. 이 서비스는 무엇을 예측하는지
사용자가 특정 콘텐츠(영화나 드라마)를 클릭할지, 혹은 시청한 후 높은 평점을 줄지 그 확률을 예측합니다.
3. 어떤 data가 사용될까?
양과 질, 다양성이 보장된 데이터를 학습용과 평가용으로 나누어 사용합니다. 사용자의 시청 기록과 검색어 같은 데이터, 콘텐츠의 장르와 배우 정보를 담은 데이터, 그리고 찜이나 좋아요 같은 데이터가 활용됩니다.
4. 모델은 어떻게 학습할까?
학습은 모델이 입력을 받아 예측값을 내놓을 때 발생하는 손실(loss)을 줄이는 과정입니다. 손실함수를 통해 계산된 오차의 기울기를 구하고 설정된 학습률만큼 가중치와 편향 등 파라미터를 수정합니다. 이 과정을 경사하강법으로 반복하며 최적의 예측 규칙을 스스로 찾아내게 됩니다.
5. 이 모델이 더 좋아지려면 어떻게 해야 할까?
사용자의 취향을 단순히 외워버리는 과적합을 방지하여 보지 않았던 새로운 장르도 적절히 추천할 수 있는 일반화 능력을 키워야 합니다. 이를 위해 시청 기록뿐만 아니라 검색 패턴이나 시청시간 등 더 다양하고 질 좋은 데이터를 추가로 확보해야 하며 학습률이나 배치 크기 같은 하이퍼파라미터를 정교하게 조절하여 추천의 정확도를 높여야 합니다.
